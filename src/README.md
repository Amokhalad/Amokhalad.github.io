## Tech Stack

HTML, CSS, and JavaScript (jQuery), with Typed.js, Waypoints, and Owl Carousel for animations and the project carousel.

## Run locally

Clone the repo and open `index.html` directly in a browser, or serve it locally:

```bash
python3 -m http.server
```

Then visit http://localhost:8000.
